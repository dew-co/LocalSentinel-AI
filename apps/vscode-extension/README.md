# LocalSentinel AI VS Code Extension

This folder prepares a future VS Code extension for LocalSentinel AI. The MVP intentionally keeps the extension as a placeholder while the web dashboard and local backend provide the core product.

Planned commands:

- LocalSentinel: Open Assistant
- LocalSentinel: Explain Current File
- LocalSentinel: Generate Fix Plan
- LocalSentinel: Send Selection to Assistant
- LocalSentinel: Analyze Project

Future behavior:

- Open a LocalSentinel AI side panel.
- Send selected code to the local FastAPI backend at `http://localhost:8000`.
- Ask Sentinel Core about the current file.
- Create a safe task plan.
- Show suggested diffs before edits.
- Connect to project memory and RAG context.
