# Continue Integration

LocalSentinel AI can work alongside Continue without claiming a full Continue plugin integration in this MVP.

Recommended workflow:

- Use Continue in VS Code with Ollama models for inline coding assistance.
- Use LocalSentinel AI for project creation, project scanning, local project memory, RAG context, sentiment prioritization, and safe action planning.
- Keep both tools pointed at local Ollama models when possible.
- Use LocalSentinel AI's dashboard for approval-first agent workflows.

Future integration options:

- Expose LocalSentinel AI as a local context provider.
- Send RAG citations from LocalSentinel AI into Continue prompts.
- Provide a local API bridge for project memory and task priorities.
- Surface safe task plans inside the future LocalSentinel VS Code extension.
